# 프로젝트 비전

## 목표

**"자연어로 Google 서비스를 제어하는 개인 AI 비서"**

Claude Code에서 자연어 명령 한 줄로 일정을 잡고, 이메일을 보내고, 파일을 관리한다.
복잡한 API 호출이나 웹 UI 조작 없이, 대화하듯 업무를 처리한다.

### 구체적 목표

1. Google Calendar + Gmail을 MCP 도구로 제어
2. Claude Code 내에서 자연어로 즉시 사용
3. 월 $10 이내 비용으로 운영
4. 점진적으로 기능 확장 (Drive, Sheets, 채팅 인터페이스, 자동화)

## 설계 원칙

### 1. 단순함 우선 (Simplicity First)

- 복잡한 프레임워크보다 명확한 코드
- 추상화 계층 최소화
- 동작하는 최소 단위부터 시작

### 2. 점진적 확장 (Incremental Growth)

- 한 번에 하나의 서비스만 추가
- 각 단계가 독립적으로 동작
- 이전 단계가 안정된 후 다음 단계로

### 3. 비용 최소화 (Cost Efficiency)

- 무료 티어 최대 활용 (Cloudflare Workers, Google API 무료 할당량)
- 불필요한 서비스 의존성 없음
- 로컬 우선, 클라우드는 필요할 때만

### 4. 투명한 기록 (Transparent Documentation)

- 모든 의사결정 기록 (ADR)
- 개발 과정 문서화
- 문서가 코드와 함께 진화
